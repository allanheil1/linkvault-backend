# Regras de contribuição (Codex)

- 1 etapa do backlog = 1 commit (no máximo 2 por etapa).
- Usar Conventional Commits: feat:, fix:, chore:, refactor:, test:, docs:.
- Antes do commit: rodar validações relevantes (build/test/lint), mostrar git status e git diff.
- Sempre aguardar meu OK antes de fazer stage e commit.
- Não realizar push automático.
